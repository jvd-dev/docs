### Request Simulação
<ResponseField name="/api/simulacao/fgts" type="POST">
```
{
	"empresa": "01",
	"agencia": "0001",
	"cpf": "00532220323",
	"lojista": "000001",
	"loja": "0001",
	"produto": "000090",
	"plano": "FG16",
	"temSeguro": false,
	"simulacaoMultipla": true
}
```
</ResponseField>

<Tip>Vale lembrar que as informações do lojista, loja, produto e plano serão disponibilizadas previamente pelo setor comercial.</Tip>
